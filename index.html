Funcionamento do Git:

Estados do Git:

Primeiros passos com o Git:

  1. Instalação:
    • apt-get install git
  2. Configurações:
    • Identidade:
      • git config --global user.name ["nomeDaPessoa"]
      • git config --global user.email [emailDaPessoa]
    • Vendo Configurações:
      • git config --list
      • git config [user.name]
  3. Ajuda:
    • git help [comando]

Trabalhando - Repositório local:

  1. Inicializando Git:
    • git init
  2. Adicionando para o próximo commit:
    • git add .
  3. Realizando commit:
    • git commit -m ["mensagem"]

Trabalhando - Repositório local (cont.):

  1. Desfazendo commit:
    • git reset --hard [checksun]
  2. Verificando o status:
    • git status
  3. Verificando registro de commits:
    • git log

  1. Criando um novo usuário.

  2. Conhecendo o sistema.
  3. Explicações:

Trabalhando - Repositório remoto:

  1. Criando um repositório.
  2. Clonando um repositório:
    • git clone [url]
  3. Enviando para o GitHub:
    • git push
